```gamemakerlanguage
scribble_font_set_default("font名もしくはsprite名の指定");
```

```gamemakerlanguage
var _element = scribble("hello,world");

// ここでエフェクトを付与してもよい

_element.draw(x,y);
```
- scribble("hello").draw(x,y)と一体に記述せずに、上記のように分けるとエフェクトを付与しやすくなる。
